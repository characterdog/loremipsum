# Lorem ipsum generator

"Lorem ipsum" is a common pseudo-Latin placeholder text used in print and web design.
It's based on "De finibus bonorum et malorum" by Marcus Tullius Cicero.

Why use this app:
* Supports 10 to 10000 words
* Material Design
* No ads or tracking
